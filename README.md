## Address Book

### Description
App for managing contacts

### Tools
Java, Swing

### Features
- CRUD of contacts
- load/save from file
- different sort options

### Build/Run
```
mvn clean compile assembly:single
cd target
AddressBook-1.0-SNAPSHOT-jar-with-dependencies.jar
```
